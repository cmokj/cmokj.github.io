用hexo,github搭建博客（中）

将博客部署到github

  1. 配置github
    进入https://github.com/settings/key
  2. 点击New SSH key输入Title&Key，但是目前我们没有Key，接着往下看
  3. 把现有的shh key都删掉
    rm -rf ~/.ssh/*
  4. 运行
    ssh -keygen -t rsa -b 4096 -C'你的邮箱'
  5. 回车四次
  6. 运行命令行得到密钥串，复制
    cat ~/.ssh/id_rsa.pub
  7. 回到第二步，将复制的密钥串填入，点击Add SSH KEY
  8. 运行命令行建立连接
    ssh -T git@github.com
  9. 如果看到Permission denied(publickey)说明你失败惹，请回到第一步大不了重新再来；否则你就成功啦～撒花～
  10. 在GitHub新建仓库，且仓库的命名必须是以下格式，下一步记得点SSH！
    你的github用户名.github.io
  11. 在本地blog目录下安装插件
    cnpm install --save hexo-deployer-git
  12. ls查看当前目录下所有文件及目录，找到_config.yml文件,用命令行打开
    vim _config.yml

  13. 最底部找到Deployment配置，进行配置

  14. 部署到远端
    hexo d

  15. 搭建成功！真棒棒！
