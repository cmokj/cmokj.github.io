用hexo,github搭建博客（上）

搭建 hexo

  1. 安装nodejs
  2. iTerm进入root用户
    sudo -i
  3. 查看版本
    node -v
    npm -v
  4. 利用npm安装cnpm
    npm install -g cnpm --registry=https://registry.npm.taobao.org
  5. 验证cnpm
    cnpm -v
  6. 用cnpm安装hexo框架
    cnpm install -g hexo-cli
  7. 验证hexo
    hexo -v
  8. 建立blog目录
  9. 初始化博客
    sudo hexo init
  10. 启动博客，用于预览
    hexo s
  11. 新建博客
    hexo n "博客名称"
  12. 编辑博客
    vim 博客名称.md
