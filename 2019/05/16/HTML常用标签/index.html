HTML常用标签

HTML常用标签

  1. HTML中特殊符号要写成转义符号
  2. <iframe>
  • 作用：在当前页面里嵌入一个页面
  • iframe可以拥有一个name，a标签的target可以通过name指向这个iframe

src也可以是相对路径

  • 现代前端开发中，iframe很少用
  1. <a>
  • 作用：跳转页面（HTTP GET请求）
  • <a href="http://qq.com" target="x">QQ</a>
    1. x= _blank，空页面打开
    2. x= _self，当前页面打开
    3. x= _parent，父页面打开，结合iframe
    4. x= _top，祖先页面打开，结合iframe
  • download属性
    <a href="http://qq/com" download>下载</a>
    是否download决定因素：
    1. content-type:application/octet-stream
    2. 在a标签中加入download属性强制下载

  • href属性

    1. href="http://qq.com"是一个使用http协议的，域名为qq.com的路径
    2. href="qq.com"是一个文件，浏览器根据文件的协议打开
    3. 写HTML时不能用file协议,比如index.html，那就无法预览，因此需要安装工具http-server，无协议绝对地址，主动继承当前页面的协议

    = 由2演变href="//qq.com"

    1. href="相对路径"
    2. href="#锚点"，不发起请求，是页面内的跳转
    3. href="?name描述"
    4. 伪协议href="javascript:;"
      点击a标签，什么也不做。用#页面会跳转，用空内容页面会刷新发送请求
  1. <form>
  • 作用：跳转页面（HTTP POST请求）
  • 提交内容会作为post请求的第四部分

  • form表单中需要提交按钮，否则无法提交表单

  • target属性

  1. <input>/<button>
  • type=submit时，为提交按钮
  • 用label标签包围描述与文本框，为一组，这样选中描述文字，即可选中文本狂
    <label>用户名：<input type="text" name="xxx"><label>
  1. <table>
