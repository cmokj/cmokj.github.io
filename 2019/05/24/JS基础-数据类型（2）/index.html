JS基础-数据类型（2）

JS基础-数据类型（2）

  1. 类型转换

1.1 转换为string

  • toSrting()
    null,undefined不可，object返回值错误
  • ‘’ + x
    null,undefined都可
    1 + ‘1’ = ‘11’;
  • window.String(x)
    全局函数，null,undefined都可

1.2 转换为布尔

  • 全局函数Boolean()
    Boolean(0); false
    Boolean(‘’); false 空字符串
    Boolean(null); false
    Boolean(undefined); false
    Boolean({});true 空数组
    Boolean(1); true
    Boolean(‘ ‘); true
  • !! x； 取反两次，结果同上

1.3 转换为number

  • Number(‘1’)===1;
  • parseInt(‘1’,10)===1; 十进制
    parseInt(‘1s1’)===1;
  • parseFloat(‘1.23’)===1.23;
  • ‘1’-0===1;
  • +’1’===1; 取正
  • -’1‘===-1；取负

2.内存图

  1. 垃圾回收

  • 如果一个对象没有被引用，它就是垃圾，将被回收。

  • IE bug:内存泄漏
    一般情况下，页面关闭，document不存在，都变成垃圾，浏览器将进行回收。
    IE6除外，不把事件标记为垃圾，导致内存资源严重浪费。

  • 深拷贝&浅拷贝
    var a=1;
    var b=a;
    #深拷贝：b改变不影响a
    #浅拷贝：b改变印象a
