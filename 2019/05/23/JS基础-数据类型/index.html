JS基础_数据类型

  1. 历史
  • 1991 李爵士 WWW
  • 1992 同事 CSS
  • 1993 W3C
  • 1995 网景公司/Netscape
    1.1 事件：发布Navigator浏览器 支持脚本
    1.2 发明者：Branden Eich，js之父
    1.3 设计时间：10天
    1.4 JS第一版：Mocha，表面上类似Java -> 改名：JavaScript
    1.5 Bug：由于Unicode,uft-8晚于js发布，所以js用了其他落后的编码方式
  • 1996 MS发布IE，仿照js发明JScript，竞争市场
    网景失败，浏览器开源（Firefox）
    为了抵制MS，向ECMA（欧洲计算机制造商协会）申报标准：ECMAScript
  • IE5.5，MS推出js发请求功能
  • 2004年，Gmail，推出网页上的程序，利用js发请求功能
    js：正式编程语言
    前端诞生：Front-end,以js为生
  • 2010左右，进入中国市场，后端->前端
  • js bug：
    都是全局变量，互相覆盖干扰（没有模块化）
    标准库缺少（内置代码少）
  • ECMAScript升级
    之前通用：ECMAScript3
    升级后：ECMAScript5（改变小）
    再次升级：ECMAScript6，成为现代编程语言
    （起因：Rail社区用ruby发明CoffeeScript，js改良版）
