HTML常用标签

HTML常用标签

  1. HTML中特殊符号要写成转义符号
  2. <iframe>
  • 作用：在当前页面里嵌入一个页面
  • iframe可以拥有一个name，a标签的target可以通过name指向这个iframe

src也可以是相对路径

  • 现代前端开发中，iframe很少用
  1. <a>
  • 作用：跳转页面（HTTP GET请求）
  • <a href="http://qq.com" target="x">QQ</a>
    1. x= _blank，空页面打开
    2. x= _self，当前页面打开
    3. x= _parent，父页面打开，结合iframe
    4. x= _top，祖先页面打开，结合iframe
  • download属性
    <a href="http://qq/com" download>下载</a>
    是否download决定因素：
    1. content-type:application/octet-stream
    2. 在a标签中加入download属性强制下载

  • href属性

    1. href="http://qq.com"是一个使用http协议的，域名为qq.com的路径
    2. href="qq.com"是一个文件，浏览器根据文件的协议打开
    3. 写HTML时不能用file协议,比如index.html，那就无法预览，因此需要安装工具http-server，无协议绝对地址，主动继承当前页面的协议

    = 由2演变href="//qq.com"

    1. href="相对路径"
    2. href="#锚点"，不发起请求，是页面内的跳转
    3. href="?name描述"
    4. 伪协议href="javascript:;"
      点击a标签，什么也不做。用#页面会跳转，用空内容页面会刷新发送请求
  1. <form>
  • 作用：跳转页面（HTTP POST请求）
  • 提交内容会作为post请求的第四部分

  • form表单中需要提交按钮，否则无法提交表单

  • target属性

  1. <input>/<button>
  • type=submit时，为提交按钮
  • 用label标签包围描述与文本框，为一组，这样选中描述文字，即可选中文本狂
    <label>用户名：<input type="text" name="xxx"><label>
  1. <table>

用hexo,github搭建博客（上）

搭建 hexo

  1. 安装nodejs
  2. iTerm进入root用户
    sudo -i
  3. 查看版本
    node -v
    npm -v
  4. 利用npm安装cnpm
    npm install -g cnpm --registry=https://registry.npm.taobao.org
  5. 验证cnpm
    cnpm -v
  6. 用cnpm安装hexo框架
    cnpm install -g hexo-cli
  7. 验证hexo
    hexo -v
  8. 建立blog目录
  9. 初始化博客
    sudo hexo init
  10. 启动博客，用于预览
    hexo s
  11. 新建博客
    hexo n "博客名称"
  12. 编辑博客
    vim 博客名称.md

用hexo,github搭建博客（中）

将博客部署到github

  1. 配置github
    进入https://github.com/settings/key
  2. 点击New SSH key输入Title&Key，但是目前我们没有Key，接着往下看
  3. 把现有的shh key都删掉
    rm -rf ~/.ssh/*
  4. 运行
    ssh -keygen -t rsa -b 4096 -C'你的邮箱'
  5. 回车四次
  6. 运行命令行得到密钥串，复制
    cat ~/.ssh/id_rsa.pub
  7. 回到第二步，将复制的密钥串填入，点击Add SSH KEY
  8. 运行命令行建立连接
    ssh -T git@github.com
  9. 如果看到Permission denied(publickey)说明你失败惹，请回到第一步大不了重新再来；否则你就成功啦～撒花～
  10. 在GitHub新建仓库，且仓库的命名必须是以下格式，下一步记得点SSH！
    你的github用户名.github.io
  11. 在本地blog目录下安装插件
    cnpm install --save hexo-deployer-git
  12. ls查看当前目录下所有文件及目录，找到_config.yml文件,用命令行打开
    vim _config.yml

  13. 最底部找到Deployment配置，进行配置

  14. 部署到远端
    hexo d

  15. 搭建成功！真棒棒！
